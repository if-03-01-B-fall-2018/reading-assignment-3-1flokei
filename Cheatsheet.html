Reading Assignment 1: Lists and Tables

Html for Lists

1. Start tags for:

  • an ordered list: ol
  • an unordered list: ul
  • a description list: dl
  • a list item: li
  • a description term: dt
  • a description data: dd

CSS for Lists

  1. Make bullet a square -> list-style-type: square;
  2. Make bullet an image named "my-bullet.png" -> list-style-image: url('my-bullet.png');
  3. Make bullet an uppercase roman number -> list-style-type: upper-roman;
  4. Set bullet position to be inside the list item's text flow -> list-style-position: inside;
  5. Set bullet position to be outside the list item's text flow -> list-style-position: outside;
